# Backend de inventario de autos 

1. Clonar repositorio
2. Ejecutar
``` npm
  pnpm install
```
1. Tener Nest CLI instalado
``` npm
  pnpm i -g @nest/cli
```
1. Levantar base de datos
``` docker
  docker-compose up
```

### Stack usado
---
- Nestjs
- MongoDB
